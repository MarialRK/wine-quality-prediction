# Wine Quality Prediction Project 
 
## Formative 1: Database - Prediction Pipeline 
 
### Project Overview 
This project implements a complete database system with SQL and MongoDB, FastAPI CRUD endpoints, and a machine learning prediction pipeline for wine quality analysis. 
 
### Features 
- **SQL Database**: 4-table schema with stored procedures and triggers 
- **MongoDB Atlas**: NoSQL implementation with collections 
- **FastAPI**: RESTful API with auto-generated documentation 
- **Prediction Pipeline**: ML prediction simulation with database logging 
 
### Tech Stack 
- Python 3.13 
- FastAPI + Uvicorn 
- SQLite + MongoDB Atlas 
- Pydantic for data validation
